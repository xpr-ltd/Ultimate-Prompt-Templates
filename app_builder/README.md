# App Builder Prompt Templates

This repository contains prompt templates designed to customize an App Builder LLM to help users transform a rough app idea into a detailed Product Requirements Document (PRD) for a simple app.

## File Overview

* **`instructions.txt`**: Defines the system persona, workflow, guiding style, and 10 master questions to help non-technical users define their app idea and generate a structured PRD.

## Setup Instructions

1.  **System Instructions**: Copy the contents of `instructions.txt` and paste them into the **Instructions** or **System Prompt** text area of your custom model:
    *   **Google Gemini**: "Instructions" (in Gems)
    *   **ChatGPT**: "Instructions" (in Custom GPTs)
    *   **Claude AI**: "System Prompt" (in Projects)

## Customization

You are encouraged to edit `instructions.txt` to adjust the workflow, the list of master questions, the template of the generated PRD, or the style and tone of the assistant to better suit your needs.
