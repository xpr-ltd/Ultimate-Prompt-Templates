# Educational Illustrator Style Guides

This repository contains visual style guides designed to maintain consistency in modern educational publishing and narrative illustrations targeted at children ages 8–14.

## File Overview

* **`mordern_narrative.md`**: Visual style guide for primary narrative storybook illustrations (warm, story-driven, culturally grounded, future-facing, with soft cinematic lighting).
* **`accent_doodle_clean_sketch.md`**: Companion visual style standard for lightweight doodles and margin sketches used in activity pages and sidebars.

## Setup Instructions

These style guides are intended to be uploaded to your custom model or referenced in your illustration workflows:

1.  **Upload Style Guides**: Download `mordern_narrative.md` or `accent_doodle_clean_sketch.md` and upload them to the knowledge/files section of your custom model:
    *   **Google Gemini**: "Knowledge" (in Gems)
    *   **ChatGPT**: "Project Sources" (in Custom GPTs)
    *   **Claude AI**: "Project Files" (in Projects)
2.  **Referencing in Prompts**: In your generation prompts, reference these files to guide the assistant in generating either a primary narrative illustration or a secondary supportive sketch.

## Customization

You are encouraged to edit both `mordern_narrative.md` and `accent_doodle_clean_sketch.md` to refine character age ranges, adjust cultural representations, or adapt the color palette for your specific book or project branding.
