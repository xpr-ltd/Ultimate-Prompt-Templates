# Minimalistic 3D Illustrator Prompt Templates

This repository contains prompt templates designed to customize an image generation assistant specializing in clean, minimalistic product-style 3D visuals.

## File Overview

* **`instructions.txt`**: Defines the system prompt, including composition rules, lighting, material guidance, and constraints required to produce modern studio tabletop 3D renders with a pure white background.

## Setup Instructions

1.  **Configure System Prompt**: Copy the contents of `instructions.txt` and paste them into the **Instructions** or **System Prompt** text area of your custom image generation model or assistant.

## Customization

You can edit `instructions.txt` to adjust surface texture defaults (e.g. from matte to glossy), default camera framing angles, lighting positions, or background setup rules (e.g., adding color or specific environments) to match your product's styling.
